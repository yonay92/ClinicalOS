# UI/UX 03 - Navigation

## Universal Search

Search any:

- Subject
- Study
- Visit
- Document
- User
- Site

## Command Palette

Shortcut:
Ctrl + K

Supports:

- Create Subject
- Create Study
- Upload Protocol
- Upload Document
- Go to Dashboard
- Open Today's Tasks

## Quick Create

Floating '+' button.

Allows rapid creation of operational records.

Navigation Rule:
Users should never navigate more than three levels deep.
