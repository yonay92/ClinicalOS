# Clinical Intelligence 09 - AI Workflows

## Protocol Workflow
Upload Protocol
→ Protocol Agent
→ Draft Study
→ Review
→ Approval

## Regulatory Workflow
Upload Document
→ Regulatory Agent
→ Metadata
→ Expiration Detection
→ Renewal Task

## Data Workflow
Completed Visit
→ Data Agent
→ Chart Priority
→ Task Queue

All workflows finish by invoking Business Rules and Audit Trail.
