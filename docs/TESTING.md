# ClinicalOS Testing Strategy

## Test Types
- Unit Tests
- Integration Tests
- End-to-End Tests
- Security Tests
- Performance Tests
- AI Validation Tests

## Critical Workflows
- Authentication
- Subject Creation
- Visit Completion
- Chart Creation
- Regulatory Upload
- Task Generation
- AI Approval
- Business Rules

## Acceptance Criteria
Every feature must:
- Pass tests
- Respect permissions
- Write Audit Trail
- Update Analytics where applicable

## Recommended Tools
- Vitest
- Playwright
- React Testing Library

## Final Rule
No feature is complete until automated tests pass.
