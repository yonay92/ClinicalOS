# UI/UX 06 - Studies

## Study List

Filters:

- Sponsor
- Site
- Therapeutic Area
- View (Active / Archived / All — defaults to Active; archived studies are
  hidden unless Archived or All is selected)

## Study Profile Tabs

- Overview
- Sites
- Subjects
- Visit Templates
- Regulatory
- Documents
- Analytics
- Timeline

## Primary Actions

- Edit Study
- Upload Protocol
- Assign Sites
- Activate Study
- Close Study
- Archive Study (never a hard delete — see BUSINESS_RULES_02_Studies.md)

AI suggestions appear in a review panel before approval.
