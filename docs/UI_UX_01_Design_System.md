# UI/UX 01 - Design System

## Purpose
Define the visual language of ClinicalOS.

## Design Principles
- Clean
- Minimal
- Enterprise
- Fast
- Accessible
- Consistent

## Color Palette
Primary: Blue
Success: Green
Warning: Amber
Danger: Red
Background: White
Surface: Light Gray

## Typography
Primary font: Inter
Headings: Bold
Body: Regular

## UI Rules
- Max 3 clicks for common actions.
- Consistent spacing.
- Reusable components only.
- No module-specific styling.
