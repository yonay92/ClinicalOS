# UI/UX 05 - Task Center

## Purpose

The Task Center is the operational home screen for every user.

## Sorting

Tasks are grouped by Site.

Within each Site:

1. 🔴 Most overdue tasks
2. 🔴 Out of Window visits
3. 🟡 Sponsor Visit related tasks
4. 🟢 Remaining work

## Task Card

- Title
- Study
- Subject
- Site
- Due date
- Priority
- Assigned user
- Quick actions

## Quick Actions

- Open Record
- Complete
- Reassign (Admin)
- Add Comment
- View History
